# ResolutionGameDevW3
My first platformer game in godot! Made during Resolution Game Design Weeek 3. Thank you Safia for the amazing tutorial and for all the help you gave me!

I learned how to make a sprite, add physics and collision rules to it as well as add my own images and text. 

Had to work on godot web, so I did track my hours with Lapse (see below) and run into some problems, but overall I like Godot!
